## **Descripción del Proyecto**
Este proyecto consiste en un programa en Java que simula un sistema de gestión de biblioteca en Java, 
aplicando los Fundamentos de [Programación Orientada a Objetos (POO)](#POO) y Herencia, utiliziando GitHub para controlar las versiones,
así como Microsoft Planner para gestión de tareas.
